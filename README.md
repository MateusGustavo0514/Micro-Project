# Micro-Project
 
A collection of small programming projects I build to practice, experiment with
ideas, and track my growth as a developer. Each project lives in its own folder,
with a README explaining what it is, what I learned from it, and how to run it.

## Repository structure
 
```
Micro-Project/
├── README.md          <- you are here
└── bookstore/
    ├── bookstore.c
    └── README.md       <- project details
```

Each folder is self-contained and includes its own README with build and run
instructions.
